FEATURED PROJECTS

OPTIMATE:

Creating a more Productive Student Experience

A university-focused web application prototype designed to help students find available library seating, organise their to-do lists and study timer. Collaboratively developed as part Women In IT's 'Future in Focus' program and awarded 3rd place for design and usefulness.

CATAN:

A simplified board game playing experience

Board games rules can be confusing for people who are new to the game, and for some players, there are many components that could cause a player to lose track of their progress. Collaboratively designed an app based on key accessibility issues that players face specifically to the board game “Catan”.

BEAR WITH US:

A simplified e-commerce purchase flow

This project aims to make improvements to the user interface design of the website Bear With Us, more specifically, its e-commerce pipeline and functions.
